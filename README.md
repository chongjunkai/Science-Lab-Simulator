# Science Lab Simulator

Science Lab Simulator is an interactive science game where students mix different chemicals to complete challenges and learn simple chemistry concepts.

## Why I Built This

I wanted to create a fun learning game that helps students explore science through experiments. Instead of only reading notes, users can choose chemicals, mix them in a virtual beaker, and see whether their combination works.

## What It Does

The game gives the player a science objective for each level, such as creating water, table salt, carbon dioxide, ammonia, methane, and other compounds. The player selects chemicals, adds them to a beaker, and clicks “Mix Chemicals” to test the answer.

## Features

* Multiple levels with different science challenges
* Chemical selection system
* Virtual beaker with liquid animation
* Mixing, bubbles, shaking, and sparkle effects
* Correct and wrong feedback
* Score system
* “Next Level” progression
* Simple chemical equations and explanations
* Clean visual design using HTML, CSS, and JavaScript
